# HTML 页面需求说明书 —— 十二个调音阶跟唱练习（最终版）

## 一、目标

开发一个**单文件 HTML 页面**，用于十二个调音阶的上行、下行跟唱练习。

特点：

* 页面标题：**音阶跟唱练习**
* 每个调一行
* 每行两个按钮：上行、下行
* 每个按钮配套一个播放进度条
* 同一时间只允许一个音频播放

## 五、功能需求

### 1️⃣ 播放与暂停

* 点击按钮：

  * 未播放 → 播放
  * 正在播放 → 暂停

### 2️⃣ 单音频播放机制

* 任何时候只允许一个音频播放
* 播放新音频时：

  * 停止旧音频
  * 旧按钮恢复 ▶

### 3️⃣ 进度条

* 使用 `<input type="range">`
* 实时同步播放时间
* 可拖动跳转播放位置

### 4️⃣ 播放完成后

* 进度条回到 0
* 按钮恢复 ▶
